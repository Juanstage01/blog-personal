Siga estos sencillos pasos para activar la pagina web

1-cree la Database "pruebas" usando el codigo en Main.php, es importante eliminar el nombre de la base de datos antes de crearlo
2-cree la tabla "bbddblog" usando el codigo en Main.php y vuelva a poner el nombre de la base de datos
3-ingrese la ruta del archivo formulario.php, para ingresar el primer post dentro de la pagina, dele click y vuelva al formulario
para ingresar a mostrar_blog.php
